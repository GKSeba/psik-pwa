# Security Policy

## Supported Versions
Projekt jest hobbystyczny. Wspierana jest tylko aktualna wersja hostowana na GitHub Pages.

## Reporting a Vulnerability
Jeśli znajdziesz problem bezpieczeństwa (np. XSS), zgłoś go przez GitHub Issues
z oznaczeniem **security** lub napisz w prywatnej wiadomości do właściciela repo.

Proszę nie publikować szczegółów podatności publicznie przed naprawą.
